Select components are used for collecting user provided information from a list of options.


``` ts
VStack(
    DropDown(item =>
        Text(item.year)
    )(selectedItem =>
        Text(selectedItem.year)
    )
    .placeholder("Some hints")
    .model(Your data)
)
```
